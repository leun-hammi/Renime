🌸 RENIME — Where Anime Meets Reality


📌 About the Project...
RENIME is a front-end web project that explores real-life Japanese locations that appeared in popular anime. The idea behind this project is to blend fiction with reality, letting users experience the real-world places behind their favorite anime stories.
This project was built as a creative UI/UX experiment while improving my skills in HTML, CSS, and Bootstrap.
This is my first website, where I used all of my understanding in front end.

✨ What the Website Does...
Presents an anime-themed landing page with user input fields
Lets users enter their name and favorite anime (UI-based interaction)
Displays a visually rich selection page of popular anime
Includes animated 3D rotating images for visual engagement
Provides a clean, card-based layout for anime selection
Uses smooth styling and transitions for a modern feel

🛠️ Technologies Used...
HTML5
CSS3
Bootstrap 4
JavaScript (basic navigation using window.open)
jQuery + Popper.js (Bootstrap dependencies)

🎨 Key Features & Highlights...
🎭 Anime-inspired UI design
🔄 3D rotating image carousel using pure CSS animations
🧭 Multi-page navigation structure
📱 Responsive layout using Bootstrap flex utilities
🖌️ Custom styling with CSS borders, gradients, and animations
💡 Simple interactive form UI (name + favorite anime input)

🧠 What I Learned...
While building RENIME, I practiced and improved my understanding of:
Structuring multi-page websites using HTML
Styling layouts with CSS Flexbox
Creating 3D animations using transform and keyframes
Integrating Bootstrap for responsive design
Managing UI flow between pages using JavaScript
Designing visually appealing front-end interfaces
